About Git

Git的学习笔记

在用hexo在GitHub上搭建博客时，安装了一个软件Git。安装它是为了把本地的网页上传到GitHub上。

Git是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理。

分布式，每个人的电脑里都有一个原版本，当修改好后，只要把改动的部分推送给对方，不需要推送完整的文件再让对象在里面找不同。

与集中式相比，集中式有中央服务器，所有的版本都储存在中央服务器中，每一次工作和修改保存，都需要在中央服务器上下载和上传，如果没有网络就不能工作。

版本控制系统，每一次修改保存都生成一个新本版，想保留旧版本要新建一个文档，而且时间长了也不记得修改了什么地方。版本控制系统就是用来管理这些版本，记录下每一次的修改过程。

GitHub网站为开源项目免费提供Git存储。
